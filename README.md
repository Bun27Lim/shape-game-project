# Controls
#### Arrow Keys - Up, Down, Left, Right. (AWSD works as well)
#### ESC        - Pause, Title Screen (Paused || End Screen), Quit game (Title Screen)
#### Q          - Rotate Counter Clockwise
#### E          - Rotate Clockwise
#### SPACE      - Place Character
#### TAB        - Start New Round
#### 1          - Toggle Sound Off/On
